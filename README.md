# Llama-Launcher-
Простое окошко с кнопками и ползунками для запуска [llama-server](https://github.com/ggml-org/llama.cpp) без консоли и без ручного набора команд. Сделано специально для тех, кто не хочет (или не умеет) разбираться с параметрами командной строки — включая детей и абсолютных новичков.
